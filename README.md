# Editable Web

Una página web moderna donde puedes editar textos e imágenes directamente en el navegador.

## Características

- ✨ Edición en tiempo real de textos e imágenes
- 💾 Guardado automático de cambios
- 🎨 Diseño moderno y responsive
- 🔥 Preparado para Firebase (colaboración en tiempo real opcional)

## Cómo usar

1. Haz clic en "Editar Página" para activar el modo edición
2. Haz clic en cualquier texto para editarlo
3. Haz clic en las imágenes para cambiarlas (subir archivo o URL)
4. Los cambios se guardan automáticamente

## Instalación Local

```bash
npm install
npm run dev
```

Este proyecto está listo para desplegarse en Vercel, Netlify u otro servicio similar.
